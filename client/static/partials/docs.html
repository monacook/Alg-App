Basic Algorithms

What are Algorithms?

So what is an algorithm? An algorithm are a set of instructions or rules specifiying how to solve some problems. For example, you're baking a cake. In order for the cake to taste like cake we need to follow the ingredients. If we don't follow the ingredients then the cake might not taste like a cake. It might give you a weird taste. Algorithms are like ingredients to follow. In order for the computer to do it's job it needs to follow instructions. And that is where we come in as software engineers. We provide the instructions, or ingredient in order for the computer to do its job. When the computer reads our code. It reads line by line until the end of it. Below is an example of an algorithm.

function sum_even(){
var sum = 0;
for(var i = 1; i < 100; i++){
if(i % 2 === 0)
sum = sum + i;
console.log(sum);
}

What's a Variable?

So let's say we have three bags and three items. One is a book, another is a watch and last, is a phone. We store the first item, which is a book and put it in the bag and label the bag. We do the same with the watch and phone and label the bag. If we have more to store we need more containers. We then close the bag. From this example, this is essentially what a variable does. A variable is a stored information that we can refer or manipulate. it can store a string, an array or an object. By looking back at the code below we set animal to equal to tiger. So now we labeled animal to equal to tiger.

var animal = "Tiger"

Comments

Comments are a wonderful thing to have. They are like side notes to your code. You can use comments for self reminders or what this certain block of code is doing. It will not interfere with your code. To comment highlight the code and if you have a mac go ahead and press command and /. Another way is using // or /* ... */. Below is an example.

function greetSomeone(person) {
if (person == "Martin") // Check whether the person is Martin
{ console.log("Hey there);
} else {
console.log("Greetings Earthling!"); //print String
}
